Fast and easy way of injecting per page Javascript and CSS code

== Description ==

This plugin will help you inject custom Javascript and CSS code into specific posts and pages. It supports all post types, even the custom ones made by other plugins / themes. Code injections are present only on those pages that you have it set up, they won't be global.
